This is a [Next.js](https://nextjs.org/) E-commerce project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).



## Snabbinstallation:

Frontend
```js
cd ecommerce-final-project
npm install
npm run dev
localhost:3000
```

Backend
```js
cd school-ecommerce
npm run dev
localhost: 3333
```
